# GestaoDeEquipamentos

- Este programa foi desenvolvido com as orientações dos professores da Academia do Programador 2025.
- Aplicado o conceito CRUD.
## As funcionalidades do programa se aplicam em:

- Registrar equipamentos

  -> Deve ter identificador único (Id);

  -> Deve ter um nome com no mínimo 6 caracteres;

  -> Deve ter um preço de aquisição;

  -> Deve ter um número de série;

  -> Deve ter uma data de fabricação;

  -> Deve ter uma fabricante;

- Vizualizar os equipamentos

	-> Deve mostrar o número;

    -> Deve mostrar o nome;

    -> Deve mostrar o preço;
 
    -> Deve mostrar a fabricante;
 
    ->  Deve mostrar a data de fabricação;

-  Controle de Chamados
 
    -> Deve ter um identificador único (Id);
 
    -> Deve ter a título do chamado;
 
    -> Deve ter a descrição do chamado;
 
    -> Deve ter um equipamento;
 
    -> Deve ter uma data de abertura;
 
    -> Deve excluir o chamado;
 
- Vizualizar os chamados

    -> Deve mostrar o título do chamado;

    -> Deve mostrar o equipamento;

    -> Deve mostrar a data de abertura;

    -> Número de dias que o chamado está aberto;

### Requisitos

- .NET SDK (recomendado .NET 8.0 ou superior) para compilação e execução do projeto.

### 1. Clone o repositório.

``` 
https://github.com/AlexAraldi/GestaoDeEquipamentos
```
### 2. Abra o terminal ou o prompt de comando e navegue até a pasta raiz

```
cd GestaoDeEquipamentos
```

### 3. Utilize o comando abaixo para restaurar as dependências do projeto.

```
dotnet restore
```

### 4. Navegue até a pasta do projeto

```
cd GestaoDeEquipamento
```
